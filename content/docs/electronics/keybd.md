title: Combo keyboard
type: docs
---

Combo keyboards combine keyboard with other inputs like mouse, joystick, media knobs, etc. 

The Ecofunctor KC1 is a combo keyboard with a joystick as the pointing device, and a knob for sound control. It aims to be compact. features include:
- 60% layout with 61 keys, including arrow keys and function keys.
- A joystick as mouse
- A small knob for volume control
- A small OLED, E-ink, or LCD display for showing information like key mapping, system status, etc.
